# CheckCompany
> 현실 목표를 달성하면서 성장하는 회사 경영 시뮬레이션 게임


### 프로젝트 소개
당신의 목표가 곧 회사의 미래입니다.

현실의 목표를 게임 속에서 달성하며 회사를 성장시켜 보세요. 실제 업무, 개인적인 도전, 심지어 취미까지! 당신의 모든 목표가 회사의 자산이 됩니다. 목표를 이룰 때마다 회사의 재정이 늘어나고, 새로운 팀원을 고용하며, 더 큰 사무실로 확장하는 성취감을 느껴보세요. 현실의 노력이 게임 속 성과로 이어지는 특별한 경험이 당신을 기다립니다.
모바일과 데스크탑 둘 다 플레이 가능합니다.

---

# 실행방법

### 코드 다운로드
```
git clone https://github.com/gestgest/CheckCompany.git
```

### 데스크탑, 모바일 다운로드
https://github.com/gestgest/CheckCompany/releases

### 기술스택
- 엔진 : 유니티
- 언어 : C#
- 서버 : Firebase
- 버전 관리 : git, github
- 에디터 : Rider
